# 更新记录

## v1.4.2（2026-07-22）

### 问题修复

- 修复退出学习页或重启应用后，“继续学习”可能从词书开头重新开始的问题
- 现在按“学习进度 × 词库”分别保存最后停留的单词
- 切换词库、返回首页或重新打开应用后，均可从上次离开的位置继续
- 数据库自动升级，保留已有词库、掌握状态、收藏和学习记录

## v1.4.1（2026-07-22）

### 问题修复

- 修复点击“下一个”时可能只在两个单词之间循环的问题
- 下一词现在严格按照当前词书顺序前进，完成一轮后自动进入下一轮
- 将按钮文字由“暂不掌握”恢复为“下一个”
- 已在 USB 连接的真机上连续验证五个单词正常前进

### 版本规则

- 修复问题增加第三位版本号，例如 `1.4.1`、`1.4.2`
- 新增或更新功能增加第二位版本号，并将第三位归零，例如 `1.5.0`
- 每次发布均递增 Android `versionCode`

## v1.4.0（2026-07-22）

### 学习流程

- 新增按词书循环的第一轮、第二轮、第三轮学习机制
- 学习页顶部以轻量固定进度条显示当前轮次、词书位置和百分比
- 标记“已掌握”后自动进入下一词
- 修复“暂不掌握”无法进入下一词的问题，并在真机验证从当前词正常切换
- 支持返回上一词并撤销误触的掌握状态

### 词条与 Excel

- 支持编辑单词、音标、词性、中英文释义和精简说明
- 兼容 `example_en` 与 `usage_brief_zh` 精简说明字段
- 导出格式改为标准 `.xlsx`
- 导出的词库可重新导入，并恢复词条、掌握状态和收藏状态
- 用户编辑过的默认词条不会被后续默认数据刷新覆盖

### 界面

- 精简背词页面，减少进度信息的视觉干扰
- 单词和释义使用柔和底色突出重点
- 精简说明不再显示单独字段标题
- 优化卡片、按钮、间距和颜色层级

### 每日打卡

- 显示日期、星期、今日学习数、今日新掌握数、轮次和词书总进度
- 随机展示一个未掌握单词及其精简说明
- 支持选择本地图片作为打卡底图
- 支持单指拖动底图、双指缩放，并保存调整位置
- 完成打卡后自动保存 PNG 到系统相册
- 支持分享到微信或其他应用

### 兼容性

- 数据库自动升级并保留已有词库、学习进度和收藏数据
- 最低支持 Android 6.0（API 23），目标 API 35
