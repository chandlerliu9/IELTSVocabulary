# 更新记录

## v1.4.0（2026-07-22）

### 学习流程

- 新增按词书循环的第一轮、第二轮、第三轮学习机制
- 学习页顶部以轻量固定进度条显示当前轮次、词书位置和百分比
- 标记“已掌握”后自动进入下一词
- 修复“暂不掌握”无法进入下一词的问题，并在真机验证从当前词正常切换
- 支持返回上一词并撤销误触的掌握状态

### 词条与 Excel

- 支持编辑单词、音标、词性、中英文释义和精简说明
- 兼容 `example_en` 与 `usage_brief_zh` 精简说明字段
- 导出格式改为标准 `.xlsx`
- 导出的词库可重新导入，并恢复词条、掌握状态和收藏状态
- 用户编辑过的默认词条不会被后续默认数据刷新覆盖

### 界面

- 精简背词页面，减少进度信息的视觉干扰
- 单词和释义使用柔和底色突出重点
- 精简说明不再显示单独字段标题
- 优化卡片、按钮、间距和颜色层级

### 每日打卡

- 显示日期、星期、今日学习数、今日新掌握数、轮次和词书总进度
- 随机展示一个未掌握单词及其精简说明
- 支持选择本地图片作为打卡底图
- 支持单指拖动底图、双指缩放，并保存调整位置
- 完成打卡后自动保存 PNG 到系统相册
- 支持分享到微信或其他应用

### 兼容性

- 数据库自动升级并保留已有词库、学习进度和收藏数据
- 最低支持 Android 6.0（API 23），目标 API 35
