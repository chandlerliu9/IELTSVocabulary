# Galaxy Store 发布清单

- 应用名称：雅思单词库
- 包名：com.tianshui.ieltsvocabulary
- 首发版本：1.0.0（versionCode 1）
- 目标 API：35
- 最低 Android：6.0（API 23）
- 发布文件：最终确认版本的 APK（GitHub Release 不发布 AAB）

## 提交前必须完成

- 注册 Samsung Account 和 Seller Portal，并申请商业卖家状态
- 确认默认词库、释义、例句、音频及应用图标拥有公开分发权
- 将隐私政策发布为公开 HTTPS 页面，并填写开发者联系邮箱
- 在真机上完整测试导入、导出、联网发音、收藏、删除自定义词库和升级保留数据
- 在 Seller Portal 中为新应用启用 Galaxy Store 应用签名并妥善保存相关密钥资料
- 准备手机截图、应用图标、支持邮箱、年龄分级和服务国家/地区
- 数据安全表中如实声明：本地学习数据不上传；缺失发音时会连接第三方网络服务

## 版本规则

后续更新必须继续使用相同包名和签名，并把 versionCode 递增为 2、3、4……。问题修复增加第三位版本号；功能更新增加第二位版本号。同一功能版本线的 GitHub Release 只保留最终确认的最新修复版。
